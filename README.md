**Project Description**

This project is a User Management System that allows CRUD operations for users.
It uses Blazor WebAssembly for the frontend and ASP.NET Core API for the backend. 


**IDE**: Microsoft Visual Studio 2022

**Tech Stack**
-----------
**Frontend**: Blazor WebAssembly 
**Backend**: ASP.NET Core 8 API
**Database**: In memory Database
